Spiel-Regeln (kurz)

  • Jeder steuert nur seine Farbe.
  • Du würfelst nur wenn du dran bist.
  • Du klickst eine eigene Figur, dann ein Ziel-Feld.
  • Barikade: genau drauf landen → aufnehmen → neues Feld wählen.
  • Bei 6: nach deinem Zug direkt nochmal würfeln.

Kurzanleitung

  1. Host: Raum leer lassen → Host drücken (Raumcode wird erzeugt).
  2. Freund: Raumcode eingeben → Beitreten.
  3. Host: wenn min. 2 Spieler online → Spiel starten.
  4. Im Zug: Würfeln, Figur klicken, Ziel klicken.
